## ASX Equity Return & Risk Analysis

**Objective**  
Analyse the historical return and risk characteristics of an ASX-listed equity using Python, with a focus on metrics relevant to investment decision-making rather than purely theoretical models.

**Scope**
- **Market:** Australian Securities Exchange (ASX)
- **Asset:** ASX-listed equity (e.g. CBA.AX)
- **Data:** Daily adjusted price data

**Methodology**
Historical price data is used to calculate key return and risk metrics, which are then interpreted from an investor-focused perspective. The analysis emphasises downside risk and volatility alongside total returns.

**Key Metrics Analysed**
- Daily and cumulative returns
- Annualised return and volatility
- Maximum drawdown
- Downside risk measures

**Tools**
- Python
- pandas
- numpy
- matplotlib
- yfinance

**Output**
- Return and volatility visualisations
- Drawdown analysis
- Summary statistics supporting risk-aware investment assessment

**Notes**
This analysis is based on historical price data and does not incorporate valuation, macroeconomic factors, or forward-looking assumptions.
